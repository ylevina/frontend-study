1. Опишите своими словами, как Вы понимаете, что такое Document Object Model (DOM)
Обьектное представление HTML документа. Где теги представленны узлами которые также являются свойствами-обьектами вложенными в главный обьект document. Ну и соответсвенно теги-узлы содержат свойства(напр аттрибуты) и вложенные элементы-свойства.
